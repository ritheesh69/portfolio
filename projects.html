Blind Assistant App

The Blind Assistant App is an AI-powered mobile application designed to help visually impaired users navigate their surroundings and perform daily tasks independently. The app uses voice commands, object detection, and text-to-speech technology to provide real-time assistance. Users can capture images using their phone camera, and the system identifies objects, text, and environmental details, then reads them aloud through speech output. The application is developed in Kotlin for Android and integrates machine learning models for object recognition and Optical Character Recognition (OCR). It also includes features such as voice-based navigation, emergency assistance, and location support. The primary goal of this project is to provide accessible technology that improves mobility, confidence, and independence for visually impaired individuals.
